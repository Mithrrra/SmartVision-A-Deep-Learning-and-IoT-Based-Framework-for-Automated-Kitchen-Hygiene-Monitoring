# SmartVision-A-Deep-Learning-and-IoT-Based-Framework-for-Automated-Kitchen-Hygiene-Monitoring
SmartVision is a real-time kitchen hygiene monitoring system that combines deep learning and IoT technologies. The system uses a YOLO-based object detection model to identify safety violations such as missing gloves, masks, and hairnets through live camera feeds. It integrates IoT sensors for environmental monitoring and provides automated alerts via email and a web dashboard.

The platform includes user authentication, live detection visualization, alert tracking, and notification systems, making it suitable for deployment in homes, restaurants, and industrial kitchens. By reducing manual inspection and enabling continuous monitoring, SmartVision enhances safety, efficiency, and hygiene compliance.
